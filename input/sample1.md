# Путешествие к звёздам

Звёздный корабль "Горизонт" медленно покидал орбиту Земли. Капитан Элена смотрела на голубую планету, которая становилась всё меньше.

- Мы готовы к прыжку? – спросил первый офицер.
- Да, – ответила капитан. – Варп-двигатель активирован.

Корабль исчез в вспышке света, и через мгновение оказался в другой галактике.

> "Космос — это последний рубеж", – вспомнила Элена слова из старого фильма.

Перед ними открылась планета, покрытая океаном из жидкого металла. Это было невероятное зрелище.

```python
def greet():
    print("Hello, alien world!")
```
Они решили отправить зонд для исследования. Миссия продолжалась.
