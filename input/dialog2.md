# AI Assistant Dialogue

User: What is a function in C?
Assistant: A function is a block of code that performs a specific task. It can take parameters and return a value.

User: Can you give an example?
Assistant: Sure! Here is a simple function that adds two numbers:

```c
int add(int a, int b) {
    return a + b;
}
```

User: How do I call it?
Assistant: You call it like this: int result = add(5, 3);

User: That's clear, thanks!
Assistant: You're welcome. Feel free to ask more questions.
