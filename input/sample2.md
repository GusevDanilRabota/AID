# Руководство по настройке сервера

## Установка

Для установки сервера выполните следующие шаги:

1. Скачайте архив с официального сайта.
2. Распакуйте в каталог `/opt/server`.
3. Запустите скрипт `install.sh`.

### Конфигурация

Файл `config.ini` содержит основные параметры:

```ini
[server]
port = 8080
host = 0.0.0.0
workers = 4
```
Не забудьте открыть порт в брандмауэре.

> Важно: Перед запуском убедитесь, что все зависимости установлены.

- Проверьте версию Python.
- Установите библиотеки: `pip install -r requirements.txt`.

После настройки выполните команду: sudo systemctl start myserver.

Теперь сервер готов к работе. Удачного администрирования!
